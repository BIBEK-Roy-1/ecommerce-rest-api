# E-commerce REST API

A RESTful E-commerce API built with Node.js and Express.js.

## Features

- User signup and signin
- JWT authentication
- Protected routes
- Product APIs
- Shopping cart APIs
- Swagger/OpenAPI documentation
- Public deployment on Render

## Tech Stack

- Node.js
- Express.js
- JavaScript
- JSON Web Token
- Swagger UI
- Git and GitHub
- Render
